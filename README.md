# Iberian Chronicles

Web Engineering I final project.

The theme is a travel blog about Spain, with pages and articles about different cities.

## Routes

- `/` - Home page
- `/about/` - About the project/blog
- `/articles/` - Articles index
- `/articles/barcelona/` - Barcelona article
- `/articles/bilbao/` - Bilbao article
- `/articles/granada/` - Granada article
- `/articles/madrid/` - Madrid article
- `/articles/seville/` - Seville article
- `/articles/toledo/` - Toledo article
- `/contact/` - Contact page

## Deployment

There is no deployment pipeline configured in this repository.

The website is live at https://iberianchronicles.nulkode.dev.

This project runs as static files (HTML, CSS, and JavaScript).

## Contact Form

The contact form works on the client side through JavaScript (`scripts/contact-form.js`) and Formspree.
